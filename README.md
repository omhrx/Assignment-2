# Console Lane Game

A simple C-based console game where the player ('O') dodges falling obstacles ('X').

### Completed Objectives
- [x] **Score System**: Real-time counter increases as obstacles pass.
- [x] **Multiple Obstacles**: Continuous stream of obstacles using an array system.
- [x] **Restart Option**: Ability to replay instantly with 'y' after Game Over.

### Controls
- **Left Arrow**: Move Left
- **Right Arrow**: Move Right
- **'y'**: Restart game (on Game Over screen)
